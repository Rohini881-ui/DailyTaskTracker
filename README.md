# Daily Task Tracker (Java)

This is a simple and beginner-friendly Java console application to manage daily tasks.  
It allows users to add tasks, view tasks, mark tasks as completed, delete tasks, and filter completed tasks.

## Features
- Add a new task
- View all tasks
- Mark a task as completed
- Delete a task by ID
- View completed tasks
- Simple and menu-driven console UI

## Technologies Used
- Java (Core Java)
- OOP Concepts
- ArrayList
